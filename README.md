# 🌐 HTML Portfolio Project

This is a simple personal portfolio website created as part of my Full Stack Development learning journey. The project showcases basic HTML skills, including linking multiple pages, adding images, and structuring a website.

---

## 📌 Project Description

This portfolio contains:

* A homepage introducing me
* A movie ranking project
* A birthday invitation project
* About Me and Contact pages

The goal of this project is to understand:

* HTML structure
* Linking multiple pages
* Using images and folders
* Basic website organization
